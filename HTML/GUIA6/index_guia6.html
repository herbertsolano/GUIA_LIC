DISCUSION DE RESULTADOS

1. Cree un editor de tablas que permita que el usuario edite los contenidos de las celdas de una tabla creada como en el ejemplo 4 del procedimiento. Para ello, además de los campos para ingresar el número de filas y columnas, debe agregar, tres campos de texto más, dos de ellos campos input de tipo number, uno para indicar la fila, otro para la columna que se va a actualizar, y el tercero donde se ingresará el dato que sustituirá a los datos de ejemplo presentes en la tabla. Además, junto al botón Crear tabla debe estar presente otro botón que servirá para actualizar el contenido de la celda que se indique. Considere la validación de los campos para ingresar la fila y columna, para que dinámicamente permitan sólo valores comprendidos entre uno y el número de filas y columnas de la tabla, respectivamente.

DISCUSION DE RESULTADOS


2. . Cree una aplicación web, similar a la del ejemplo 2 del procedimiento de esta guía, en donde el usuario pueda seleccionar entre tres tipos de combustible (regular, especial y diésel) haciendo uso de un campo de formulario de tipo radio button. El precio de cada tipo de combustible está establecido de la siguiente forma: la gasolina especial tiene un costo de $4.27, la gasolina regular un costo de $4.05 y el diésel un costo de $3.96. Además, debe haber un campo de tipo number que permita un valor mínimo de 0.05 galones y un máximo de 150 galones. Además, el tamaño de paso en las cantidades de galones debe ser de 0.05. Cuando el usuario indique el tipo de combustible que se servirá y la cantidad de galones que se servirá (en variaciones de 0.05 ctvos.) debe actualizarse otro campo de texto de sólo lectura donde el usuario podrá ver el monto a pagar por el tipo y la cantidad de gasolina que se va a servir.
